02

Typography字型

All-sans: Inter (EN) + Noto Sans TC (ZH). Verified from tsmc.com — Nimrod MT is logo-only, not a web font. Hierarchy is weight-driven (700 for display, 400 for body). Tabular numerals throughout. 全無襯線字體：英文採 Inter，中文採 Noto Sans TC。已對照 tsmc.com 確認 — Nimrod MT 僅限 logo，不作為網頁字型。層級以字重區分（700 為標題，400 為內文）。所有數字皆等寬。
